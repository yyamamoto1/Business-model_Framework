# ユーザーストーリー (記入例)

**As a** 忙しい中小企業のマネージャー,
**I want to** チームメンバーの今日のタスク一覧をワンクリックで確認したい,
**so that** 朝会で素早く進捗を確認し、次の指示を出せる。

---

## 受け入れ基準 (Acceptance Criteria)

- [ ] ダッシュボードに「チームの今日のタスク」ウィジェットが存在する。
- [ ] ウィジェットには、全チームメンバーの、今日が期限のタスクが表示される。
- [ ] タスクは担当者ごとにグループ化されている。
- [ ] 完了したタスクは、表示から除外されている。
